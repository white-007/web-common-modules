# 便利小二-供应商服务系统
## Project setup
```
npm install

如果报错：unable to resolve dependency tree则执行 npm i --legacy-peer-deps
```

如果速度过慢可以使用淘宝源下载：

```
npm install --registry=https://registry.npm.taobao.org
```

### Compiles and hot-reloads for development

npm run serve
